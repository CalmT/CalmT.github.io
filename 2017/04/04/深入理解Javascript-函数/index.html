深入理解Javascript-函数

发表于 2017-04-04 | 阅读次数

函数的定义

函数使用function关键字来定义，可以用函数定义表达式和函数声明语句来定义函数。

函数声明语句

函数声明包括三部分：
函数名称是必需部分，函数名称就相当于变量名，新定义的函数对象赋值给这个变量；
圆括号包含的0或n个参数的参数列表，这些参数就相当于函数体中的局部变量；
花括号包含的0或n条javascript语句构成的函数体，一旦函数被调用就会执行这些语句。


    function funcName (args,...){
        ...
    }

函数定义表达式

函数定义表达式包括三部分：
可选的函数名称，如果函数定义表达式包含函数名称，函数的局部作用域将会包含一个绑定到函数对象的名称，即函数名称为函数内的一个局部变量；
圆括号包含的0或n个参数的参数列表，这些参数就相当于函数体中的局部变量；
花括号包含的0或n条javascript语句构成的函数体，一旦函数被调用就会执行这些语句。


    var variable = function [funcName] (args,...){
        ...
    }

函数返回值

大多数函数会包含一条return语句，return语句导致函数停止执行，并返回它的表达式的值给函数调用者；
如果只是一个return语句但没有表达式，返回undefined给调用者；
如果函数没有return语句，则只执行函数体中的语句，并返回undefined给调用者。

两种函数定义方式的区别：

1、函数声明语句可以出现在全局代码中，或内嵌在其他函数中，但不能出现在循环、条件判断，或者try/catch/finally以及with语句中；
而函数定义表达式能出现在Javascript代码任何地方。
2、函数声明语句被提前到外部函数作用域或外部脚本的最顶端，故可以在函数声明语句之前调用声明的函数；而要使用函数表达式定义的函数，必须将表达式赋值给一个变量，变量的声明会提前，赋值却不会提前，故不能再函数表达式定义的函数定义之前调用函数。

函数的调用

构成函数体的javascript代码在函数定义时不会执行，只有调用该函数的时候才会执行。
可以通过以下四种方法调用Javascript函数：
1、作为函数调用
2、作为方法调用
3、作为构造函数调用
4、通过函数的apply()和call()方法间接调用

函数调用

使用函数调用表达式进行普通函数调用，一个函数调用表达式由多个函数表达式组成，每个函数表达式由函数对象和圆括号包含的参数列表组成；
ECMAScript3和非严格模式下的ECMAScript5中，调用上下文是全局对象，严格模式下，调用上下文是undefined；

方法调用

方法是保存在对象属性里的Javascript函数，作为方法的调用，调用上下文为调用这个方法的对象。

构造函数调用

如果函数或方法调用之前带有new关键字，就构成构造函数调用；
如果构造函数调用时传入了实参列表，则与函数调用和方法调用一致，先计算实参列表，再传入函数内；如果构造函数没有形参，则构造函数调用可以省略圆括号；
构造函数调用创建一个新的空对象，并以这个新对象作为调用上下文，即构造函数可以以this来引用新创建的对象；

间接调用

Javascript中函数也是对象，可以包含方法，其中apply()和call()可以用来简介调用函数，都允许显式指定this的值，使任何函数都可以作为任何对象的方法来调用
